## Overview
Blazor Server web portal for the Internal Developer Platform. Provides a Copilot-style chat interface, conversation history, and campaign management views. Communicates with `idp-agents` via HTTP using OBO (On-Behalf-Of) flow.

**Tech Stack**: ASP.NET Core 9, Blazor Server, Fluent UI, Cosmos DB, Entra ID, Terraform, GitHub Actions
